PA!GO

Googleとの共同開発による知育デバイス。子どもが外の世界で出会った花、動物、虫などを内蔵カメラで撮影すると、Googleの機械学習技術（TensorFlow、Edge TPU）によりAIがそれらが何かを音声で教えてくれる。スマホやゲーム機の代わりに子どもが野外で過ごす時間を増やし、「探索する」「遊ぶ」「学習する」意欲を育むことを目的とする。

リミックス

設計・開発

レトロ家電ハックプロジェクト。1962年発売のパナソニック製キャビネット家具調ステレオ（スーパーフォニックステレオHE-3000）を現代のテクノロジーでアップデート。レコード針のかわりにカラーセンサーを搭載し、レコード盤状のシートに描かれた絵を音に変えて音楽を奏でる楽器に生まれ変わらせた。色に音階を割り当て思い通りのメロディを奏でることも、自由な絵で予想不可能な音楽を奏でることも可能。過去の作り手との対話を通じて、デジタルとフィジカルの交差する未来への問いを形にしたプロジェクト。

Project Page

言山百景

テクニカルディレクション

人と地域の新たな関わり方による地域活性プロジェクト。リモート観光やオンラインイベントの普及により希薄化していく「人と地域との関係性」をつなぎ変える新たな魅力の引き出し方を提案。スマートフォンで発した言葉が遠く離れた土地に"物理的に"舞い降りるオブジェとなり、その地域に新しい魅力や文化を生み出すネットワーク連動型インスタレーション。

Project Page

他力研究所

テクニカルディレクション

目に見えない神秘的な力（他力）の計測・採取を行い、その活用方法を探求する研究所チーム。従来の産業において誤差や障害と見なされ排除されてきた自然の不確定要素を、共に活かすべき存在として捉え直し、「揺らぎ」や「偶然」と手を取り合って未来の産業のあり方を再構築することを目的とする実験的研究。

Project Page

EPYTOTORP

テクニカルディレクション

ユーザーとメーカーが一緒にプロトタイプ（完成前の試作品）をブラッシュアップするコミュニティプラットフォーム。プロトタイプ名は「PROTOTYPE」を逆から読んだもの。ソフトウェア開発のオープンソースコミュニティの考えを応用し、消費者自身がContributor（貢献者）としてものづくりに参加・貢献することで、より素晴らしいプロダクトに磨かれる新しい製品開発手法を提案。

Project Page

D+IO

テクニカルディレクション

人間が本来備え持っている創造力をエンパワーし、「大切な誰かへ届けたいみんなのモノづくり」を応援するプロジェクト。これまでの「DIY = Do It Yourself」から「Do It Ourselves」へ発想を転換。市販の電子部品や身の回りの素材を使ったプロダクトの作り方をGitHubで公開し、ユーザー間でソースコードをカスタマイズしたりアップデートを提案し合ったりできる仕組みを提供。

Project Page

KOTOBATABI

テクニカルディレクション
Red Dot Award: Design Concept 2022

みんなの言葉が雲になり、偶発的な発見を誘発するインタラクティブツール。音声認識により発話した言葉がテキスト化され、現実と紐付いたデジタル空間に「言葉雲」として舞い上がるARアプリケーション。スマホやARグラスを通じて他のユーザーの言葉雲を見ることができ、GPS内蔵の専用音声入力デバイスでは言葉雲がある場所で光ってお知らせ。言葉雲が集まっている場所を発見して新たな出会いや発見を求める「ちょっとした旅」を誘発する体験を提供する。
